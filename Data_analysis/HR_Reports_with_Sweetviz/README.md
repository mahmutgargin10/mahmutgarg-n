# HR Data Analytics Project 📊

This project focuses on a comprehensive **Human Resources (HR) Data Analysis** using Python and the **Sweetviz** library. The analysis explores key workforce metrics including salary distributions, performance evaluations, and attendance patterns.

## 🚀 Analysis Highlights
- **Exploratory Data Analysis (EDA):** Automated visualization of all variables.
- **Key Metrics:** In-depth look at "Absences," "Salary," and "Performance Scores."
- **Correlation Study:** Identifying how engagement levels affect overall productivity.
